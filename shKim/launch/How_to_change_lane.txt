add car_test.launch, kobiki2.launch.xml file to deu_car/launch

if you want located turtlebot at first lane 
run this command.
roscd deu_car && source ./gazebo_env.sh && roslaunch deu_car car_test.launch

or if you want located turtlebot at second lane
run this command.
roscd deu_car && source ./gazebo_env.sh && roslaunch deu_car car_test2.launch

It needs some time to change location. So wait at least 30 second after gazebo process killed and run another lane's command.

-------------------------------------------------------------------------------------------------------------------------------

차선을 변경하기 위해서 car_test.launch, kobiki2.launch.xml 파일을 deu_car/launch에 추가한다.

터틀봇을 1차선으로 시작하는 명령어는
roscd deu_car && source ./gazebo_env.sh && roslaunch deu_car car_test.launch
이고

터틀봇을 2차선으로 시작하는 명령어는
roscd deu_car && source ./gazebo_env.sh && roslaunch deu_car car_test2.launch
이다.

차선 변경 시 가제보 프로세스 종료 후 30초 이상 대기하였다가 
다른 차선의 명령어를 입력해야 올바르게 터틀봇이 해당 차선으로 이동한다.
